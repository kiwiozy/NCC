# Letterhead Export Guide (Adobe Illustrator)

1. Open `Walk Easy_Letterhead.ai`
2. Make sure Artboard is **A4 (210 × 297 mm)**.
3. Export TWO versions:

## A. Print
- File → Save As → PDF
- Preset: High Quality Print
- Size: A4
- Color: CMYK
- Resolution: 300 ppi
- Name: `Walk-Easy_Letterhead-PRINT.pdf`

## B. Web / App
- File → Export → Export As → PNG
- Resolution: 300 ppi
- Background: Transparent (if your text area is white)
- Name: `Walk-Easy_Letterhead-WEB.png`
- Place this at: `frontend/public/letterhead.png`

4. Do **not** change the white writing box size. The app uses:
- top: 60mm
- left: 22mm
- right: 18mm
- bottom: 45mm
